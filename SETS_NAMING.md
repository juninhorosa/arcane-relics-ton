# Nomenclatura dos Sets — 20 Grades × 5 Classes

Cada grade representa um tier de poder. Dentro de cada grade, existem 5 conjuntos (um por classe), cada um com 5 itens (Arma, Capacete, Armadura, Luvas, Botas).
